# Ozzy for Business - Backend

Ozzy for Business is an AI-powered, chat-based financial platform for small business owners in Africa. 

This repository contains the FastAPI backend, which handles transaction parsing (NLP), accounting reports, and payment integrations.

## Features
- **AI Transaction Parsing**: Understands natural language like "Sold dress 20k".
- **Accounting Engine**: Generates P&L, Balance Sheet, and Cash Flow reports.
- **Payment Integration**: Supports MTN Uganda, Airtel Uganda, and Flutterwave.
- **Automated Summaries**: Daily, weekly, and monthly business insights generated by AI.

## Tech Stack
- **Framework**: FastAPI
- **Database**: PostgreSQL (via SQLAlchemy)
- **AI**: Google Gemini 1.5 Flash
- **Task Scheduling**: Cron-based background scripts

## Setup

1. **Install Dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

2. **Environment Variables**:
   Copy `.env.example` to `.env` and fill in the required keys.

3. **Run Migrations**:
   ```bash
   alembic upgrade head
   ```

4. **Start the Server**:
   ```bash
   uvicorn app.main:app --reload
   ```
